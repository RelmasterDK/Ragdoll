Наша программа выравнивает изображение номера, которое вы в неё загрузите!
Все библиотеки, которые мы использовали:
cv2 (OpenCV): Мощная библиотека для обработки изображений (чтение, преобразование цвета, контраст, контуры).
numpy: Численные вычисления для работы с массивами и матрицами.
typing: Аннотации типов для улучшения читаемости и структурирования кода.
skimage (scikit-image): Алгоритмы обработки изображений (оттенки серого, нахождение краев, преобразование Хафа, поворот).
matplotlib: Визуализация данных и изображений.
itertools: Эффективная обработка данных (группировка результатов распознавания текста).
glob: Поиск файлов по шаблону.
tensorflow: Модель распознавания текста с использованием нейронных сетей (TFLite).
requests, io: HTTP-запросы и работа с потоками ввода-вывода.
Так же мы использовали нейросеть, которая определяет границы номеров.

Запустить проект очень легко! Просто запустите exe файл и выберите изображение после нажатия кнопки в окне. Далее выйдет изображение слева до и справа после всех преобразований.
